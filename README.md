# Facededup Android SDK — `ng.facededup:facededup`

Drop-in face-liveness + identity verification for Android (managed `WebView` flow,
typed `FacededupResult`). `minSdk 24`. Bundled offline MediaPipe engine.

- Maven (tokenless): `https://swiftend-assets-348761024048.s3.eu-west-2.amazonaws.com/m2`
  → `implementation("ng.facededup:facededup:0.5.0")`
- Modules: `:facededup` (public SDK), `:liveness` (engine).
- Docs: https://facededup.ai/docs#android

Extracted from the Facededup mono-repo (`facededup/nablr-liveness`). The sample app
moved to `facededup/demos`.
